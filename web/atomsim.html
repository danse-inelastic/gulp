AtomSim

AtomSim is a Web-deployed interface to forcefield-based atom dynamics programs. Currently it is an applet and supports the well-known General Lattice Utility Program (GULP). In the future it will become a rich internet application and will support additional Engines such as MMTK. It can be accessed by clicking the launch button below. Java 1.5 or higher must be installed on your machine.

Features:

  • atomic viewer and table editor

  • job launcher and remote job submission handler
